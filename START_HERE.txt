🚀 【快速开始 - 3种方式任选】
═══════════════════════════════════════════════════════════════

方式1️⃣: 自动化一键训练（最简单）⭐⭐⭐⭐⭐
────────────────────────────────────────────
Windows: 双击 RUN_AUTO_TRAIN.bat
Linux:   python auto_train.py

✨ 特点: 自动安装、自动数据、使用最好的模型(YOLOv8x)


方式2️⃣: 图形化菜单（推荐）⭐⭐⭐⭐
────────────────────────────────────────────
Windows: 双击 start.bat
Linux:   bash start.sh

✨ 特点: 交互式菜单，可选择训练/推理/验证等操作


方式3️⃣: 命令行手动控制（高级）⭐⭐⭐
─────────────��──────────────────────────────
python train.py --data data.yaml --model yolov8x.pt --epochs 100

✨ 特点: 完全自定义参数

💻 【系统要求】
═══════════════════════════════════════════════════════════════

最低: Python 3.8+, 8GB RAM
推荐: Python 3.10+, NVIDIA GPU (8GB显存), CUDA 11.8+


📂 【准备数据】（可选 - 有演示数据）
═══════════════════════════════════════════════════════════════

如果有自己的数据:
   raw_data/
   ├── images/      ← 放你的图像
   └── labels/      ← YOLO格式标签 (.txt文件)

标签格式: <class_id> <x_center> <y_center> <width> <height>

然后运行自动脚本，会自动检测并使用你的数据！

📊 【训练完成后】
═══════════════════════════════════════════════════════════════

结果位置: runs/train/exp/weights/best.pt  ← 这是你的最佳模型

推理测试:
   python inference.py --weights runs/train/exp/weights/best.pt --source test.jpg

实时检测:
   python inference.py --weights runs/train/exp/weights/best.pt --camera 0

导出部署:
   python train.py --export onnx --weights runs/train/exp/weights/best.pt
